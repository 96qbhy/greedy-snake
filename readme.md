# greedy-snake
贪吃蛇小游戏

## 运行游戏
```shell
git clone https://github.com/96qbhy/greedy-snake.git
cd greedy-snake
npm install
npm start
```
> 默认运行在 http://localhost:8000

## 玩法
1. 方向  
w => 上,  
d => 右,  
s => 下,  
a => 左,   
space(空格) => 加速,最高5速。
